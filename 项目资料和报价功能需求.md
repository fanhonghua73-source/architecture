# 项目资料和报价功能需求

## 一、项目资料扩展

### 1. 用地红线
- **格式**: PDF文件上传
- **功能**: 
  - 上传用地红线PDF
  - 在线预览
  - 下载原文件
  - **AI识别**: 自动识别PDF中的红线，生成CAD格式

### 2. 规划条件
- **格式**: PDF/Word文档
- **内容**: 规划局出具的规划条件文件
- **功能**: 上传、预览、下载

### 3. 甲方确认函
- **格式**: PDF/Word/图片
- **内容**: 甲方签字确认的文件
- **功能**: 上传、预览、下载

### 4. 百度位置
- **功能**: 
  - 输入项目地址
  - 自动调用百度地图API定位
  - 显示项目在地图上的位置
  - 保存经纬度坐标
  - 地图截图保存

### 5. 特殊规范
- **格式**: PDF/Word文档
- **内容**: 项目特殊要求的规范文件
- **功能**: 上传、预览、下载、分类管理

## 二、报价数据扩展

### 原有数据
- 项目名称
- 项目地址
- 建筑面积
- 项目预算

### 新增数据
1. **总建筑面积** (㎡)
   - 地上建筑面积
   - 地下建筑面积
   - 总计

2. **户数**
   - 住宅户数
   - 商业户数
   - 其他户数

3. **配套面积** (㎡)
   - 商业配套
   - 公共配套
   - 其他配套

4. **地下室预估面积** (㎡)
   - 地下车库
   - 地下设备用房
   - 其他地下空间

## 三、PDF红线转CAD功能

### 技术方案

#### 方案一：AI识别（推荐）
1. 上传PDF红线图
2. 使用AI视觉模型识别红线
3. 提取坐标点
4. 生成DXF格式CAD文件
5. 下载CAD文件

#### 方案二：手动标注
1. 上传PDF红线图
2. 在线标注工具标记红线
3. 系统记录坐标点
4. 生成CAD文件

### 实现步骤
1. PDF转图片
2. 图像处理（边缘检测、线条识别）
3. 坐标提取
4. CAD格式生成（DXF）

## 六、API接口设计

### 项目资料接口
```
POST   /api/projects/{id}/documents/upload        # 上传文档
GET    /api/projects/{id}/documents               # 获取文档列表
GET    /api/projects/{id}/documents/{doc_id}      # 下载文档
DELETE /api/projects/{id}/documents/{doc_id}      # 删除文档
POST   /api/projects/{id}/land-boundary/to-cad    # PDF转CAD
```

### 百度地图接口
```
POST   /api/projects/{id}/location                # 设置位置
GET    /api/projects/{id}/location                # 获取位置
POST   /api/projects/{id}/location/geocode        # 地址转坐标
```

### 报价数据接口
```
PUT    /api/projects/{id}/quotation               # 更新报价数据
GET    /api/projects/{id}/quotation               # 获取报价数据
POST   /api/projects/{id}/quotation/export        # 导出报价单
```

## 十、注意事项

1. **文件大小限制**: PDF文件建议不超过50MB
2. **CAD精度**: 转换精度取决于PDF质量
3. **地图API**: 需要申请百度地图API Key
4. **存储空间**: 文档较多时注意存储空间
5. **权限控制**: 敏感文档需要权限控制
