# 灵感自动导入功能使用说明

## 功能概述

灵感自动导入功能允许你从指定文件夹批量导入灵感图片及其配置信息，大大提高了灵感管理的效率。

## 使用步骤

### 1. 准备文件夹

创建一个专门用于存放灵感的文件夹，例如：
```
F:\architecture\inspirations_import
```

### 2. 准备文件

在文件夹中放置图片和对应的txt配置文件：

```
inspirations_import/
├─ 现代建筑.jpg          # 图片文件
├─ 现代建筑.txt          # 配置文件（与图片同名）
├─ 室内设计.png
├─ 室内设计.txt
├─ 景观设计.jpg
└─ 景观设计.txt
```

### 3. 配置文件格式

每个txt文件包含以下信息（使用冒号或中文冒号分隔）：

```
标题: 现代建筑外观设计
描述: 采用玻璃幕墙和简约线条的现代建筑设计
分类: 外观
风格: 现代
标签: 玻璃幕墙,简约,现代主义
```

**支持的字段：**
- `标题` 或 `title`: 灵感标题
- `描述` 或 `description`: 详细描述
- `分类` 或 `category`: 外观/室内/景观/细节
- `风格` 或 `style`: 现代/古典/工业风等
- `标签` 或 `tags`: 多个标签用逗号分隔

### 4. 配置导入路径

1. 登录root账号
2. 进入"我的" -> "灵感自动导入"
3. 输入文件夹路径：`F:\architecture\inspirations_import`
4. 点击"保存配置"

### 5. 执行导入

点击"立即导入"按钮，系统会：
1. 扫描指定文件夹中的所有图片
2. 读取对应的txt配置文件
3. 将图片复制到系统上传目录
4. 在数据库中创建灵感记录

### 6. 查看结果

导入完成后会显示：
- 成功导入的灵感数量
- 如有错误，会列出失败的文件

## 支持的图片格式

- JPG / JPEG
- PNG
- GIF
- WEBP

## 注意事项

1. **文件命名**：图片和txt文件必须同名（扩展名除外）
2. **编码格式**：txt文件必须使用UTF-8编码
3. **权限要求**：只有root用户可以配置和执行导入
4. **文件大小**：建议单个图片不超过10MB
5. **路径格式**：Windows系统使用反斜杠或正斜杠都可以

## 常见问题

### Q: 如果没有txt文件会怎样？
A: 系统会使用图片文件名作为标题，其他字段为空。

### Q: 可以重复导入吗？
A: 可以，每次导入都会创建新的灵感记录。

### Q: 导入后原文件会被删除吗？
A: 不会，原文件保持不变，系统只是复制图片。

### Q: 如何批量修改已导入的灵感？
A: 目前需要在灵感库中逐个编辑，后续版本会支持批量操作。

## 技术支持

如有问题，请检查：
1. 文件夹路径是否正确
2. 文件权限是否足够
3. txt文件编码是否为UTF-8
4. 后端服务是否正常运行
