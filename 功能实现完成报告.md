## 🔧 API接口

### 新增接口
```
# 认证相关
POST   /api/auth/change-password           # 修改密码
POST   /api/auth/avatar                    # 上传头像
GET    /api/auth/users                     # 获取所有用户（root）
PUT    /api/auth/users/{id}/approve        # 审批用户（root）
PUT    /api/auth/users/{id}                # 编辑用户（root）

# 合同相关
POST   /api/contracts/ai-config            # 添加AI配置
GET    /api/contracts/ai-config            # 获取AI配置
POST   /api/contracts/generate             # 生成合同
GET    /api/contracts/{id}                 # 获取合同详情

# 权限相关
POST   /api/permissions                    # 分配项目权限
GET    /api/permissions/project/{id}       # 获取项目权限

# 灵感导入
GET    /api/inspirations/auto-import-config      # 获取导入配置
POST   /api/inspirations/set-auto-import-config  # 设置导入配置
POST   /api/inspirations/import-from-folder      # 执行导入
```

## 📝 使用说明

### Root用户功能
1. **用户审批**: 我的 -> 用户管理 -> 审批新用户

5. **上传头像**: 我的 -> 点击头像
