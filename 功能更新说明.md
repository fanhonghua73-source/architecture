# 建筑师助手 - 功能更新说明

## 新增功能

### 1. 邮箱格式验证
- ✅ 注册时自动验证邮箱格式
- ✅ 格式错误时弹出提示

### 2. 个人信息管理
- ✅ 编辑个人信息页面（邮箱、手机、公司、职位）
- ✅ 修改密码功能
- ✅ 头像上传功能（点击头像即可更换）

### 3. Root用户系统
- ✅ 默认root用户（用户名：root，密码：root123456）
- ✅ 用户注册需要root审批
- ✅ Root用户可以管理所有用户信息
- ✅ Root用户可以编辑其他用户信息
- ✅ Root用户可以指定用户编辑项目权限

### 4. 合同生成功能
- ✅ 支持MiniMax和Gemini AI接口
- ✅ Root用户可配置AI接口
- ✅ 根据项目信息和合同模板生成合同
- ✅ 支持自定义合同模板

### 5. 权限管理
- ✅ 项目权限分配系统
- ✅ Root用户可指定用户编辑特定项目
- ✅ 灵感库和项目的编辑权限控制

## 使用说明

### 初始化Root用户

首次使用需要创建root用户：

```bash
cd backend
python create_root.py
```

默认账号信息：
- 用户名：root
- 密码：root123456
- 邮箱：root@architect.com

**请登录后立即修改密码！**

### Root用户功能

登录root账号后，在"我的"页面可以看到：
1. **用户管理** - 审批新用户、编辑用户信息
2. **AI配置** - 配置MiniMax或Gemini接口
3. **生成合同** - 使用AI生成建筑设计合同

### 普通用户注册流程

1. 用户注册账号
2. 等待root用户审批
3. 审批通过后才能登录使用

### 头像上传

在"我的"页面点击头像，选择图片即可上传。

### 修改个人信息

在"我的"页面点击"编辑个人信息"，可修改：
- 邮箱
- 手机号
- 公司
- 职位

### 修改密码

在"我的"页面点击"修改密码"，输入旧密码和新密码即可。

## API接口更新

### 认证相关
```
POST   /api/auth/change-password    # 修改密码
POST   /api/auth/avatar             # 上传头像
GET    /api/auth/users              # 获取所有用户（root）
PUT    /api/auth/users/{id}/approve # 审批用户（root）
PUT    /api/auth/users/{id}         # 编辑用户（root）
```

### 合同相关
```
POST   /api/contracts/ai-config     # 添加AI配置（root）
GET    /api/contracts/ai-config     # 获取AI配置（root）
POST   /api/contracts/generate      # 生成合同（root）
GET    /api/contracts/{id}          # 获取合同详情
GET    /api/contracts/project/{id}  # 获取项目合同列表
```

### 权限相关
```
POST   /api/permissions             # 分配项目权限（root/所有者）
GET    /api/permissions/project/{id} # 获取项目权限列表
DELETE /api/permissions/{id}        # 删除权限
```

## 数据库更新

新增字段：
- `users.is_root` - 是否为root用户
- `users.is_approved` - 是否已审批

新增表：
- `project_permissions` - 项目权限表
- `contracts` - 合同表
- `ai_configs` - AI配置表

## 注意事项

1. **安全性**：请务必修改root默认密码
2. **AI接口**：需要有效的MiniMax或Gemini API Key
3. **权限控制**：只有root用户可以审批用户和配置AI
4. **数据迁移**：如果已有数据库，需要重新创建或手动添加新字段

## 测试流程

1. 运行 `python create_root.py` 创建root用户
2. 启动后端：`python main.py`
3. 启动前端
4. 使用root账号登录
5. 配置AI接口（在"我的" -> "AI配置"）
6. 注册一个普通用户
7. 使用root账号审批该用户
8. 测试合同生成功能
