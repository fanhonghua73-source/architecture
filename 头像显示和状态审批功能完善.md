### 2. 项目状态变更审批 ✅
**需求**：项目创建者修改项目状态（进行中/已完成/暂停）需要root审批

**实现内容**：

#### 数据库
- 新增`ProjectStatusChange`表记录状态变更申请
  - `old_status`: 原状态
  - `new_status`: 新状态
  - `requested_by`: 申请人
  - `approval_status`: 审批状态（pending/approved/rejected）
  - `approved_by`: 审批人
  - `reason`: 变更原因

#### 后端API
```python
POST   /api/projects/{id}/request-status-change    # 申请状态变更
GET    /api/projects/status-changes/pending        # 获取待审批列表
POST   /api/projects/status-changes/{id}/approve   # 审批状态变更
```

#### 前端逻辑
- **Root用户**：点击状态直接修改
- **项目创建者**：点击状态提交审批申请
- **其他用户**：无法修改状态

### 3. 合同模板上传 ✅
**需求**：生成合同时支持上传Word/PDF模板

**实现内容**：
- 添加模板选择方式：文本输入 / 文件上传
- 支持上传Word(.doc, .docx)和PDF(.pdf)文件
- 文件上传后显示文件名
- 支持删除已上传文件

## 头像URL处理逻辑

```javascript
const getAvatarUrl = (avatar) => {
  if (!avatar) {
    return '/static/default-avatar.png'  // 默认头像
  }
  if (avatar.startsWith('http')) {
    return avatar  // 完整URL直接返回
  }
  return `${config.baseURL}${avatar}`  // 相对路径拼接baseURL
}
```

## 使用说明

### 项目创建者修改状态
1. 进入项目详情页
2. 点击状态标签（进行中/已完成/暂停）
3. 选择新状态
4. 系统提示"需要管理员审批"
5. 提交申请
6. 等待root审批

### Root审批状态变更
1. 进入"我的" -> "项目审批"（待添加入口）
2. 查看待审批的状态变更列表
3. 点击"批准"或"拒绝"
4. 项目状态自动更新

### 上传合同模板
1. 进入"生成合同"页面
2. 选择"上传文件"选项
3. 点击上传按钮
4. 选择Word或PDF文件
5. 文件上传成功后显示文件名
6. 填写其他信息后生成合同
