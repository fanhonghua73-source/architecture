# 快速启动指南

## 方式一：自动启动（推荐）

### 启动后端

```bash
cd backend
chmod +x start.sh
./start.sh
```

### 启动前端

使用 HBuilderX 打开 `frontend` 目录，然后运行到微信开发者工具或浏览器。

---

## 方式二：手动启动

### 1. 启动后端

```bash
# 进入后端目录
cd backend

# 安装依赖（首次运行需要）
pip3 install fastapi uvicorn sqlalchemy pydantic pydantic-settings python-jose passlib python-multipart aiofiles pillow python-dotenv

# 或者使用 requirements.txt
pip3 install -r requirements.txt

# 启动服务
python3 main.py
```

后端服务将在 http://localhost:8000 启动

**验证后端是否启动成功：**
- 访问 http://localhost:8000 应该看到欢迎信息
- 访问 http://localhost:8000/docs 查看API文档

### 2. 启动前端

#### 方式A：使用 HBuilderX（推荐）

1. 下载并安装 HBuilderX：https://www.dcloud.io/hbuilderx.html
2. 打开 HBuilderX
3. 文件 -> 打开目录 -> 选择 `frontend` 目录
4. 修改 `config.js` 中的 `baseURL`（如果后端不是 localhost:8000）
5. 运行 -> 运行到浏览器 -> Chrome（或运行到微信开发者工具）

#### 方式B：使用命令行

```bash
# 进入前端目录
cd frontend

# 安装依赖（首次运行需要）
npm install

# 运行到H5
npm run dev:h5

# 或运行到微信小程序
npm run dev:mp-weixin
```

## 测试流程

### 1. 测试后端API

使用浏览器访问 http://localhost:8000/docs，你会看到 Swagger UI 界面。

**测试注册和登录：**

1. 找到 `POST /api/auth/register` 接口
2. 点击 "Try it out"
3. 输入测试数据：
```json
{
  "username": "test_user",
  "email": "test@example.com",
  "password": "test123456",
  "phone": "13800138000",
  "company": "测试公司",
  "position": "建筑师"
}
```
4. 点击 "Execute" 执行
5. 如果返回 200，说明注册成功

6. 找到 `POST /api/auth/login` 接口
7. 输入：
```json
{
  "username": "test_user",
  "password": "test123456"
}
```
8. 执行后会返回 `access_token`，复制这个token

9. 点击页面右上角的 "Authorize" 按钮
10. 输入：`Bearer {你的token}`（注意Bearer后面有空格）
11. 点击 "Authorize"

现在你可以测试其他需要认证的接口了！

### 2. 测试前端

1. 启动前端后，会自动打开浏览器或微信开发者工具
2. 首次进入会看到登录页面
3. 点击"立即注册"，填写信息注册
4. 注册成功后，使用用户名和密码登录
5. 登录成功后会进入首页，可以看到：
   - 用户信息
   - 数据统计
   - 快速操作入口
   - 最近项目列表

6. 测试各个功能：
   - 点击"新建项目"创建一个项目
   - 切换到"灵感"标签，添加一些设计灵感
   - 切换到"材料"标签，查看材料库
   - 切换到"我的"标签，查看个人信息

## 常见问题

### 1. 后端启动失败

**问题：** `ModuleNotFoundError: No module named 'fastapi'`

**解决：** 依赖未安装，运行：
```bash
pip3 install -r requirements.txt
```

**问题：** `Address already in use`

**解决：** 端口8000被占用，修改 `main.py` 最后一行的端口号：
```python
uvicorn.run("main:app", host="0.0.0.0", port=8001, reload=True)
```

### 2. 前端无法连接后端

**问题：** 网络请求失败

**解决：** 
1. 确认后端已启动（访问 http://localhost:8000）
2. 检查 `frontend/config.js` 中的 `baseURL` 是否正确
3. 如果是微信小程序，需要在开发者工具中关闭"不校验合法域名"

### 3. 图片上传失败

**问题：** 上传图片时报错

**解决：**
1. 确认图片大小不超过10MB
2. 确认图片格式为 jpg/jpeg/png/gif/webp
3. 确认后端 `uploads` 目录有写入权限

### 4. Token过期

**问题：** 提示"未授权，请重新登录"

**解决：** Token默认有效期24小时，过期后重新登录即可

## 开发建议

### 后端开发

1. 修改代码后，uvicorn会自动重启（热重载）
2. 查看日志了解请求详情
3. 使用 Swagger UI 测试接口
4. 数据库文件：`architect_assistant.db`
5. 上传文件目录：`uploads/`

1. 使用 HBuilderX 的代码提示功能
2. 使用 Chrome DevTools 调试
3. 微信小程序使用微信开发者工具调试
4. 修改代码后会自动刷新
