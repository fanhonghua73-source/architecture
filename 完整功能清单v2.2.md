## 🛠️ 技术栈

- FastAPI

- JWT认证

- Python-docx (Word解析)

### 前端

- uni-ui组件库

### 数据库
- SQLite (开发)
- PostgreSQL (生产推荐)

## 📦 部署清单

### 环境要求
- Python 3.8+
- Node.js 14+
- 数据库（SQLite/PostgreSQL）

### 依赖安装
```bash
cd backend
pip install -r requirements.txt
```

### 数据库初始化
```bash
python create_root.py
```

```bash

python main.py

使用HBuilderX运行
```
