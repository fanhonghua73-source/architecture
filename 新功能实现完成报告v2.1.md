## 🎯 使用说明

### 1. AI生图
1. 点击首页"AI生图"
2. 上传基础图片（必需）
3. 上传参考图片（可选，最多3张）
4. 输入提示词（可选）
5. 点击"生成设计图"
6. 查看结果，可以：
   - 重新生成（调整提示词）
   - 保存到灵感库

### 2. 项目批量导入
1. 准备txt或md文件，格式如下：
```
---
项目名称: 现代办公楼
地址: 北京市朝阳区
面积: 5000
预算: 20000000
状态: 进行中
描述: 现代化办公空间设计
---
```
2. 在项目管理页面上传文件
3. 系统自动解析并创建项目

### 3. 合同生成PDF
1. 进入"生成合同"页面
2. 选择项目
3. 上传Word/PDF模板或输入文本
4. 选择AI提供商
5. 点击生成
6. 下载PDF合同

### 4. 项目权限管理
1. Root用户进入项目详情
2. 查看"项目权限管理"区域
3. 点击"添加用户"
4. 选择用户和权限类型
5. 用户可以管理被授权的项目

### 5. 项目图片查看
1. 进入项目详情
2. 图片以轮播方式展示
3. 左右滑动查看所有图片
4. 点击图片可放大预览
