# 安装和启动说明

## 问题诊断

当前遇到网络连接问题，无法从 PyPI 下载包。

## 解决方案

### 方案一：检查网络连接

```bash
# 测试网络连接
ping pypi.org

# 如果无法连接，可能需要：
# 1. 检查网络设置
# 2. 配置代理
# 3. 使用国内镜像源
```

### 方案二：使用国内镜像源（推荐）

```bash
cd "/Users/jiangyige/Documents/it学习笔记/lian/backend"

# 使用清华镜像源安装
pip3 install -i https://pypi.tuna.tsinghua.edu.cn/simple fastapi uvicorn sqlalchemy pydantic pydantic-settings python-jose passlib python-multipart aiofiles pillow python-dotenv --user

# 或者使用阿里云镜像
pip3 install -i https://mirrors.aliyun.com/pypi/simple/ fastapi uvicorn sqlalchemy pydantic pydantic-settings python-jose passlib python-multipart aiofiles pillow python-dotenv --user
```

### 方案三：配置永久镜像源

创建或编辑 `~/.pip/pip.conf` 文件：

```bash
mkdir -p ~/.pip
cat > ~/.pip/pip.conf << EOF
[global]
index-url = https://pypi.tuna.tsinghua.edu.cn/simple
[install]
trusted-host = pypi.tuna.tsinghua.edu.cn
EOF
```

然后再次安装：
```bash
cd "/Users/jiangyige/Documents/it学习笔记/lian/backend"
pip3 install -r requirements.txt --user
```

### 方案四：逐个安装依赖

## 常见问题

### 1. 端口被占用

如果提示端口8000被占用，修改 `backend/main.py` 最后一行：
```python
uvicorn.run("main:app", host="0.0.0.0", port=8001, reload=True)  # 改成8001
```

同时修改 `frontend/config.js`：
```javascript
baseURL: 'http://localhost:8001'  // 改成8001
```

### 2. 权限问题

如果遇到权限问题，可以尝试：
```bash
# 不使用 --user 参数
pip3 install -r requirements.txt

# 或者使用虚拟环境
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python main.py
```

### 3. Python版本问题

确保Python版本 >= 3.8：
```bash
python3 --version
```

如果版本太低，需要升级Python。
